프로젝트 소개

#오늘 뭐 먹지? (2022.3~2022.4)

-7人 팀 프로젝트
-사이트 주소 : http://itwillbs9.cafe24.com/Team_1
코드리뷰 : kny8451.tistory.com/category/TeamProject
GITHUB : github.com/kny0304/Team_1

개인의 레시피를 공유하고, 식자재도 구매할 수 있는 웹 사이트 입니다.
1인가구가 증가함에 따라 하루하루 무엇을 먹어야 할지 고민하는 사람들이 늘어나고 있는 시대상을 반영하여 '오늘 뭐 먹지?'라는 이름을 짓게 되었습니다.
사이트를 방문해 여러 레시피를 둘러보고 마음에 드는 레시피를 저장할 수도 있으며 신선한 재료를 구매할 수 있습니다.

기능 소개

주요기능 : 회원 / 식품판매 / 레시피작성 / 고객센터 / 마이페이지(장바구니, 구매, 게시글 좋아요리스트)

<내가 구현한 기능>
1. 장바구니 목록 : 로그인한 회원의 id(PK)값으로 현재 장바구니에 담겨있는 상품을 보여줍니다. 수량변경, 삭제가 가능합니다.

2. 주문정보 작성 : 주문을 위한 배송정보를 입력하고, 주문 상품 확인을 할 수 있습니다. AJAX를 이용해 주문자 정보와 동일한 정보를 클릭 한번으로 입력할 수 있습니다.

3. 구매내역 확인 : 각 주문건마다 주문번호를 부여해 과거 주문했던 목록을 확인할 수 있습니다.

사용 기술 : JAVA / SPRING FRAMEWORK / HTML / CSS / JAVASCRIPT / JQUERY / MYSQL / AJAX

FEEDBACK

수업시간에 배웠던 것을 실제로 적용해 보며 머리속에 어지럽게 펼쳐져 있던 이론들을 어느정도 이해할 수 있는 과정이었습니다. 기본 기능(CRUD)을 포함하여 내가 생각한 기능을 실제로 구현해 가는 과정이 흥미로웠고 몇날 며칠이 걸려 구현에 성공했을 때의 성취감이 매우 좋았습니다.

그러나 처음으로 진행해본 팀 프로젝트인 만큼 아쉬운 점이 많이 남은 프로젝트입니다. 시간이 부족해 시작 단계에서 계획했던 기능들의 70% 정도만 구현되었습니다. 제가 맡은 부분에서는 장바구니의 각 상품을 선택하여 주문/삭제하는 기능, 주문 내역 리스트를 조회할 때 주문한 상품중 대표상품의 썸네일만 보이도록 하는 기능 등이 추가로 필요합니다.
